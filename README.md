# 关于ouster

ouster是一个用Go语言写的mmorpg。这个项目是服务端部分，目前还在此项目还处在早期开发阶段。

客户端部分在[这里](https://github.com/tiancaiamao/ouster-client)。

# 开发计划

以最小代价基于[flare](https://github.com/clintbellanger/flare-engine)的客户端做出一个mmorpg（原型)。

packet相关的处理
场景内行走/移动
游戏逻辑移进服务端
